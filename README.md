# SSH Arena

Многопользовательская терминальная RTS‑игра с элементами градостроительства и симуляции жизни.

Игроки подключаются по SSH, видят мир в ASCII‑графике и управляют развитием своего поселения: строят здания, добывают ресурсы, заботятся о жителях, исследуют технологии, торгуют, воюют с соседями.

## Технологический стек

- **Язык**: Go 1.26+
- **Конфигурация**: Viper (YAML, env vars)
- **Логирование**: Zerolog (структурированные логи)
- **Сеть**: SSH (golang.org/x/crypto/ssh)
- **Хранение данных**: SQLite (github.com/mattn/go-sqlite3)
- **События**: Внутренняя шина событий (pub/sub)
- **Рендеринг**: ANSI/ASCII через собственный движок

## Архитектура

Проект построен по принципам чистой архитектуры (Clean Architecture) с разделением на слои:

1. **Домен (Domain)** – интерфейсы и бизнес‑логика (`internal/interfaces.go`)
2. **Ядро (Core)** – игровой движок (`internal/engine/`)
3. **Инфраструктура (Infrastructure)** – реализация сетевого транспорта, хранилища, рендерера (`internal/network/`, `internal/storage/`, `internal/ui/`)
4. **Вспомогательные пакеты (pkg/)** – конфигурация, логгер, утилиты

## Быстрый старт

### Требования

- Go 1.26 или новее
- SSH‑клиент (для подключения игроков)

### Установка

```bash
git clone <repository>
cd ssh-arena-app
go mod download
```

### Конфигурация

Отредактируйте `configs/config.yaml` под свои нужды (порт SSH, размер мира и т.д.).

### Запуск сервера

```bash
go run cmd/server/main.go
```

Сервер начнёт слушать SSH‑подключения на порту 2222 (по умолчанию) и HTTP‑метрики на порту 8080.

### Подключение игрока

```bash
ssh -p 2222 localhost
```

После аутентификации игрок увидит ASCII‑интерфейс и сможет начать игру.

## Разработка

### Добавление новой функциональности

1. Определите интерфейс в соответствующем пакете `internal/`.
2. Реализуйте интерфейс в инфраструктурном слое.
3. Внедрите зависимость через конструктор в `cmd/server/main.go`.
4. Напишите unit‑тесты (см. `*_test.go`).

### Логирование

Используйте глобальный логгер из `pkg/logger`:

```go
import "ssh-arena-app/pkg/logger"

log := logger.Get()
log.Info("message", "key", "value")
```

### Конфигурация

Все настройки загружаются через `pkg/config`. Добавьте новые поля в структуры `Config` и `config.yaml`.
